# 旅游信息管理系统 java41

## 项目概述

旅游信息管理系统是一个基于 Spring Boot 和 Vue 的前后端分离的系统。它包含了丰富的功能模块，旨在提供一站式的旅游信息服务。

## 技术栈

- **前端语言**：Vue
- **后端语言**：Java, Spring Boot
- **主要技术**：Java, Spring Boot, MyBatis, MySQL, Vue

## 功能模块

### 前台功能

- 展示旅游新闻
- 景区信息浏览
- 美食信息展示
- 展示旅游线路
- 在线留言

### 系统角色

#### 后台管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

#### 普通用户

- 查看旅游线路
- 查看订单信息
